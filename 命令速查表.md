# Git 命令速查表

## 第一次配置

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
git config --list
```

## 把普通文件夹变成 Git 仓库

```bash
git init
git status
```

## 提交代码

```bash
git status
git add 文件名
git add .
git commit -m "提交说明"
```

## 查看历史

```bash
git log
git log --oneline
git log --oneline --graph --decorate --all
```

## 查看差异

```bash
git diff
git diff --staged
```

## 撤销与回退

```bash
# 丢弃某个文件在工作区的修改，慎用
git restore 文件名

# 取消已经 add 的文件
git restore --staged 文件名

# 回退到上一个提交，但保留修改在工作区
git reset --soft HEAD~1

# 回退到上一个提交，并取消暂存，但保留文件修改
git reset --mixed HEAD~1

# 回退到上一个提交，连文件修改也丢掉，危险
git reset --hard HEAD~1

# 生成一个反向提交，推荐用于已经 push 到远程后的回退
git revert 提交ID
```

## 分支

```bash
git branch
git switch -c feature/login
git switch main
git merge feature/login
git branch -d feature/login
```

## 远程仓库

```bash
git remote -v
git remote add origin 远程仓库地址
git push -u origin main
git pull
git fetch
```

## 同时连接 GitHub 和 Gitee

```bash
git remote add github GitHub仓库地址
git remote add gitee Gitee仓库地址
git push -u github main
git push -u gitee main
```

以后更新：

```bash
git push github main
git push gitee main
```
