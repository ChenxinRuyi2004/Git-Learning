# Git / GitHub / Gitee 学习仓库

这个文件夹是你的 Git 项目管理练习场。

建议学习顺序：

1. 先读 [Git-GitHub-Gitee-从0到上手.md](D:/Codex/Git%20Learning/Git-GitHub-Gitee-从0到上手.md)
2. 边读边在终端敲命令
3. 遇到忘记的命令，看 [命令速查表.md](D:/Codex/Git%20Learning/命令速查表.md)
4. 在 [exercises](D:/Codex/Git%20Learning/exercises) 文件夹里完成练习

你需要记住的 Git 核心流程：

```text
工作区修改文件 -> git add -> git commit -> git push
远程有更新 -> git pull
想看状态 -> git status
想看历史 -> git log --oneline --graph
```
